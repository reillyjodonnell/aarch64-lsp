### Server

`bun run server`

### Client

1. `cd client && bun run dev` (watches for changes)
2. `code .` (opens client in new window)
3. F5 to launch extension host
4. Open an `.aarch64` file in the new window

Note when updating client or server, be sure to reload the window from the client
